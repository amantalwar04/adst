# Reference Guides These publications are available in PDF format. ## Downloading Adobe Acrobat Reader To view PDF documents within the help system, you must have Adobe Acrobat Reader and the Acrobat Web plug-in properly installed in your browser. You can download a copy from the [Adobe Reader download website](https://get.adobe.com/reader/). ## Viewing and printing PDF files Click any of the following links to view or print any of the PDF files in the following list: - [UG643 - OS and Libraries Document Collection](http://www.xilinx.com/cgi-bin/docs/rdoc?v=replace;d=oslib_rm.pdf) - [UG953 - Vivado Design Suite 7 Series FPGA and Zynq-7000 All Programmable SoC Libraries Guide](http://www.xilinx.com/cgi-bin/docs/rdoc?v=replace;d=ug953-vivado-7series-libraries.pdf) - [UG1043 - Embedded System Tools Reference Manual](http://www.xilinx.com/cgi-bin/docs/rdoc?v=replace;d=ug1043-embedded-system-tools.pdf)
