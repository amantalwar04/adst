# Zynq DRAM Diagnostics Test The Zynq-7000 AP SoC DRAM diagnostics test is a stand-alone program running on a single Zynq-7000 AP SoC-7000 AP SoC Cortex-A9 processor, executing out of OCM. The program uses the UART for interactive operation.
