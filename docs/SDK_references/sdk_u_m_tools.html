# Xilinx Tools Menu The **Xilinx Tools** menu can be accessed from the Xilinx SDK menubar. The table below lists the items available on the **Xilinx Tools** menu. |Name |Function | |------|----------| |[Generate Linker Script](../SDK_concepts/concept_generatelinkerscript.html)|Configure and generate the linker script.| |[Board Support Package Settings](../SDK_concepts/concept_bspsettings.html)|Customize the Board Support Package created for application development within SDK, including libraries and drivers, or for external tools.| |[Repositories](../SDK_concepts/sdk_c_repos.html)|Add, change, or remove software repositories.| |[Program FPGA](sdk_u_program_fpga.html)|Program the FPGA. \(Keyboard shortcut: **Ctrl+Shift+D**\)| |[Dump/Restore Data File](../SDK_concepts/concept_dump_restoredatefile.html)|Copy the memory file contents to a data file and restore data file contents back to memory.| |[Program Flash](sdk_u_program_flash.html)|Program a Flash memory on the target board.| |[Launch Shell](../SDK_concepts/concept_launchshell.html)|Launch the Command shell.| |[XSCT Console](../SDK_concepts/concept_xsct_console.html)|Open the XSCT Console| |[Configure QEMU Settings](../SDK_concepts/concept_configure_qemu_settings.html)|Specify configuration and debug settings for QEMU. **Note:** This option is available only in the Linux version of Xilinx SDK. | |[Create Boot Image](../SDK_concepts/concept_create_zynq_bootimage.html)|Use to create a Zynq bootable image.| -
