# gdbremote connect Connect to GDB remote server. ## Syntax gdbremote connect \[options\] server Connect to a GDB remote server, for example qemu. A special client named tcfgdbclient is used to connect to remote GDB server. ## Options |Option|Description| |------|-----------| |-architecture |Specify default architecture is remote server does not provide it.| ## Returns Nothing, if the connection is successful.
