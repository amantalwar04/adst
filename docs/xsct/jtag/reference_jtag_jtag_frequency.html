# jtag frequency Get/set JTAG frequency. ## Syntax jtag frequency Get JTAG clock frequency for current scan chain. jtag frequency -list Get list of supported JTAG clock frequencies for current scan chain. jtag frequency frequency Set JTAG clock frequency for current scan chain. ## Returns Current Jtag frequency, if no arguments are specified, or if Jtag frequency is successfully set. Supported Jtag frequencies, if -list option is used.
