# version Get SDK or TCF server version. ## Syntax version \[options\] Get SDK or TCF server version. When no option is specified, SDK build version is returned. ## Options |Option|Description| |------|-----------| |-server|Get the TCF server build version, for the active connection.| ## Returns SDK or TCF Server version, on success.
