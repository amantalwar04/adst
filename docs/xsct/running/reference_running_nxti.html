# nxti Step over a machine instruction. ## Syntax nxti \[count\] Step over a single machine instruction. If instruction is function call, execution continues until control returns from the function. If `` is greater than 1, repeat `` times. Default value of count is 1. ## Returns Nothing, if the target has stepped to the next address. Error string, if the target is already running or cannot be resumed.
