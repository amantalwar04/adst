# rwr Write to register ## Syntax rwr Write the `` to active target register specified by `` For a processor core target, processor core register can be written to. For a target representing a group of processor cores, system registers or IOU registers can be written. ## Returns Nothing, if successful. Error string, if an invalid register is specified or the register cannot be written. ## Example\(s\) ``` rwr r8 0x0 ``` Write 0x0 to register r8. ``` rwr usr r8 0x0 ``` Write 0x0 to register r8 in group usr. **Parent topic:** [Target Registers](../../xsct/registers/reference_xsct_registers.html) **Related information** [rrd](../../xsct/registers/reference_registers_rrd.html)
