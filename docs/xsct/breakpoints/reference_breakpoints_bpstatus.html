# bpstatus Print Breakpoint/Watchpoint status. ## Syntax bpstatus Print the status of a Breakpoint/Watchpoint specified by ``. Status includes the target information for which the Breakpoint is active and also Breakpoint hitcount or error message. ## Options None ## Returns Breakpoint status, if the breakpoint exists.
