# createhw Create a hardware project. ## Syntax createhw \[OPTIONS\] Create a hardware project using a hardware specification file. ## Options |Option|Description| |------|-----------| |-name |Project name that should be created.| |-hwspec |Hardware specification file for creating a hardware project.| ## Returns Nothing, if the hardware project is created successfully. Error string, if invalid options are used or if the project cannot be created. ## Example\(s\) ``` createhw -name hw1 -hwspec system.hdf ``` Create a hardware project with name hw1 from the hardware
