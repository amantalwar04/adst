# getperipherals Get a list of all peripherals in the HW design. ## Syntax getperipherals Return the list of all the peripherals in the hardware design, along with version and type. If \[processor-instance\] is specified, return only a list of slave peripherals connected to that processor. ## Options None ## Returns If successful, this command returns the list of peripherals. Otherwise it returns an error. ## Example\(s\) ``` getperipherals system.hdf ``` Return a list of peripherals in the hardware design. ``` getperipherals system.hdf ps7_cortexa9_0 ``` Return a list of peripherals connected to processor CortexA9\#0 in the hardware design. **Parent topic:** [SDK Projects](../../xsct/sdk/reference_xsct_sdk.html) **Related information**
