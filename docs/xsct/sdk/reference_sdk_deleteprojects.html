# deleteprojects Delete project\(s\) from the workspace. ## Syntax deleteprojects \[OPTIONS\] Delete project\(s\) from the workspace or disk. ## Options |Option|Description| |------|-----------| |-name|Project name/list to be deleted List of projects should be separated by semi-colon \{proj1;proj2;proj3\}.| |-workspace-only|Delete project from workspace only and not from disk. Default operation is to delete projects from disk.| ## Returns Nothing, if the projects are deleted successfully. Error string, if invalid options are used or if the project cannot be deleted. ## Example\(s\) ``` deleteprojects -name hello1 ``` Delete the hello1 project from the disk. ``` deleteprojects -name hello1 -workspace-only ``` Delete the hello1 project from workspace only. **Parent topic:** [SDK Projects](../../xsct/sdk/reference_xsct_sdk.html) **Related information**
