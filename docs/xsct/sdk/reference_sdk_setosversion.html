# setosversion Set OS version in BSP. ## Syntax setosversion \[OPTIONS\] Set specified OS version in the BSP. Latest version is added by default. ## Options |Option|Description| |------|-----------| |-bsp |BSP project or mss file corresponding to the OS.| |-ver |Version of the OS. Default latest version of OS is added.| ## Returns If successful, this command returns nothing. Otherwise it returns an error. ## Example\(s\) ``` setosversion -bsp hello_bsp -ver 5.4 ``` Set the OS version 5.4 in the BSP. ``` setosversion -bsp hello_bsp ``` Set the latest OS version from repo
