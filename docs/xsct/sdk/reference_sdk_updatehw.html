# updatehw Update a hardware project. ## Syntax updatehw \[OPTIONS\] Update a hardware project with the changes in the new hardware specification file. ## Options |Option|Description| |------|-----------| |-hw |Hardware project that should be updated.| |-newhwspec |New hardware specification file for updating the hardware project.| ## Returns Nothing, if the HW project is updated successfully. Error string, if invalid options are used or if the project cannot be updated. ## Example\(s\) ``` updatehw -hw hw1 -newhwspec system.hdf ``` Update the hardware project hw1 with the changes in the
