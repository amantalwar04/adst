# setlib Set library and version in BSP. ## Syntax setlib \[OPTIONS\] Add a library to BSP. If version is not specified, latest library version available is added. If library is already available in BSP, the library version is updated. ## Options |Option|Description| |------|-----------| |-bsp |BSP project or mss file.| |-lib |Library name to be added to BSP.| |-ver |Library version to be added. Default latest version of library is added.| ## Returns If successful, this command returns nothing. Otherwise it returns an error. ## Example\(s\) ``` setlib -bsp hello_bsp -lib xilffs -ver 2.0 ``` Add xilffs library version 2.0 to the BSP. ``` setlib -bsp hello_bsp -lib xilrsa ``` Add latest version of xilrsa library available in repo, to the BSP. **Parent topic:** [SDK Projects](../../xsct/sdk/reference_xsct_sdk.html) **Related information**
