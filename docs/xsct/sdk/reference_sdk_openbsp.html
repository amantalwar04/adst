# openbsp Open the BSP. ## Syntax openbsp Open the BSP from BSP project created using 'createbsp', or from the MSS file. ## Options None ## Returns If successful, this command returns nothing. Otherwise it returns an error. ## Example\(s\) ``` openbsp hello_bsp ``` Open the BSP project 'hello\_bsp'. ``` openbsp /tmp/wrk/hello_bsp/system.mss ``` Open the bsp project corresponding to the
