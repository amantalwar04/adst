# changebsp Change the referenced BSP for an application. ## Syntax changebsp \[OPTIONS\] Change the referenced BSP for an application project. ## Options |Option|Description| |------|-----------| |-app |Application project whose BSP should be changed.| |-newbsp |New BSP which the application should refer.| ## Returns Nothing, if the Application Project refernece changed. Error string, if invalid options are used or if the project cannot be updated. ## Example\(s\) ``` changebsp -app app1 ``` Lists the bsp projects which can be referenced by the given application project changebsp -app app1 -newbsp test2\_bsp.
