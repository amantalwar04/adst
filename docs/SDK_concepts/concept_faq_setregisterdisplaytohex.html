# How can I set the register display format so that it defaults to "Hexadecimal"? To change the default register format: 1. Select **Window \> Preferences.** 2. Open the tree for C/C++ in the list on the left-hand side. 3. Click **Debug** in the expanded tree. 4. In the Default Register Format drop-down list, select **Hexadecimal.** **Parent topic:** [Frequently Asked Questions \(FAQs\)](../SDK_concepts/sdk_debugging_faq.html)
