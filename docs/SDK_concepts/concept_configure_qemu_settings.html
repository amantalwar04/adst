# Configure QEMU Settings You can use the **Configure QEMU Settings** dialog box to configure and debug your applications on QEMU. **Note:** Debugging applications using QEMU is not supported on WindowsÂ®. ![](../images/configure_QEMU.png) The table below lists all the options available on the Configure QEMU Settings dialog box: |Option|Description| |------|-----------| |Advanced settings|Select the option to specify the path to the QEMU script \(.SH\) and the GDB port number.For example: `/tmp/qemu-system-aarch64 -nographic -M arm-generic-fdt -dtb /tmp/zcu102-arm.dtb -gdb tcp::1137 -S`| |QEMU executable path| Path to the QEMU executable. If not specified, the default QEMU executable available at /bin/qemu-system-aarch64 folder is used. | |Device tree blob path| Path to the device tree blob \(DTB\).
