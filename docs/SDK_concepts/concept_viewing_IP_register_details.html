# Viewing IP Register Details XilinxÂ® SDK now supports viewing of IP register details, using either the [Hardware \(system.hdf\)](#viewipreghw) view or during debug using the [Registers view](#viewipregview). ## Viewing IP register details using the Hardware \(system.hdf\) view After a successful hardware project creation, SDK opens the system.hdf file in the **Editor** view. The file now displays cross-references to the registers of IP blocks present in the design. ![](../images/view_ip_register_details.png) To view register details, click on the **Registers** link on the system.hdf file in the **Editor** view. ![](../images/view_ip_register_field_details.png) ## Viewing IP register details using the Registers view Debug a hardware project and launch the **Registers** view to view the registers and their field values. ![](../images/view_ip_register_registerview.png) You can modify editable field values, during debug.
