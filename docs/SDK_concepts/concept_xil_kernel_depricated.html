# Xil\_Kernel depricated warning message is issued when I create an application or edit BSP settings. Xil\_Kernel is depricated and will not be available in current and future Xilinx SDK releases. If you are using a repository from previous builds which contain Xil\_Kernel, warning message appears when an application is created or a BSP is modified. **Parent topic:** [Frequently Asked Questions \(FAQs\)](../SDK_concepts/sdk_debugging_faq.html)
