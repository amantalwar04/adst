# XSCT Console Xilinx Software Command-line Tool \(XSCT\) is an interactive and scriptable command-line interface to Xilinx Software Development Kit \(Xilinx SDK\). As with other Xilinx tools, the scripting language for XSCT is based on Tools Command Language \(Tcl\).
