# How do I specify the Reset options to be used when a Debug \(or Run\) session is launched? You can specify the Reset options in the Initialization tab of the Debug Launch Configuration dialog box. The following options can be specified: - System reset: To reset the entire system - Processor reset: To reset a specific processor - No Reset: To not reset anything **Parent topic:** [Frequently Asked Questions \(FAQs\)](../SDK_concepts/sdk_debugging_faq.html)
