# Configuring a Board Support Package \(External Tools\) To configure an existing Board Support Package \(BSP\) project created for use with external tools, do the following: 1. In the SDK Project Explorer or C/C++ Navigator pane, right-click the BSP you want to configure. 2. Select **Board Support Package Settings.** The Board Support Package Settings dialog box opens for configuring a BSP for external tools. **Note:**You can also open this dialog box by selecting **Xilinx Tools \> Board Support Package Settings.** 3. In the Board Support Package Settings dialog box, configure parameters for your OS. 4. When you finish with all the settings you want to make, click **OK** to commit your changes. 5. If the **Build Automatically** option is selected in the **Project** menu, your BSP is regenerated with your new settings applied. ![](../images/sdk_t_config_bsp_external-1.png) **Parent topic:** [Board Support Packages \(External Tools\)](../SDK_concepts/sdk_c_bsp_external.html) **Related information** [Creating a Board Support Package \(External Tools\)](../SDK_tasks/sdk_t_create_bsp_external.html) [Generating a Board Support Package for External Tools](../SDK_tasks/sdk_t_build_bsp_external.html)
