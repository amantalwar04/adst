# Limitations - Importing from already existing BIF file is supported in Read-only mode. You cannot modify or edit the contents of a BIF file after importing it. - Multiline comments are not supported. For example: ``` /* Comments – 1 Comments – 2 */ ``` **Parent topic:** [Creating a Boot Image](../SDK_tasks/task_creatingabootimage.html) **Related information** [Assumptions](../SDK_tasks/task_assumptions.html)
