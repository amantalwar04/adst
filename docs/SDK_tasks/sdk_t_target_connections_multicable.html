# Target Connections View To view all the cable-types and devices, connected to the host, using the **Target Connections** view, do the following: 1. Select **Window \> Show View \> Xilinx \> Target Connections** to launch the **Target Connections** view. 2. Double-click the target connection. The **Target Connection Details** dialog box appears. 3. Click **Advanced**. A list of active cable-types and devices appear in the **Target Connection Details** dialog box. ![](../images/Target_connection_details_multicable.png) **Parent topic:** [Multi-Cable and Multi-Device Support](../SDK_references/reference_multicable_multidevice.html)
