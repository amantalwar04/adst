# Creating an OpenCV Template Application To create an OpenCV template application: 1. Select **New \> Application Project.**The New Project wizard opens and displays the Application Project page. ![](../images/sdk_t_opencv_template-1.png) 2. In the **OS Platform** drop-down box, select **Linux**. 3. In the **Processor Type** drop-down box, select the processor of any ARM core. 4. Select the **C++** radio button under Target Software. 5. Click **Next**. The Templates page opens. 6. Select the template type **OpenCV Example Application.** **Note:** The **OpenCV Example Application** option is available only if you select the **C++** Language option in the Target Software pane. ![](../images/sdk_t_opencv_template-2.png) 7. Click **Finish**. SDK creates an OpenCV example application. You can edit and change this application as required. 8. You can add additional header files. To do this: 1. Right-click on the project name and select **C/C++ Build Settings.**The Properties dialog box opens, with the C/C++ Build \> Settings pane selected. ![](../images/cc_build_settings_db.png) 2. Add header files by clicking the **Add**button:![](../images/icon_device_add.gif) 3. Exclude header file paths by clicking the **Delete** button:![](../images/icon_device_delete.gif) 4. Add library paths in the Libraries settings page. ![](../images/sdk_t_opencv_template-6.png) **Parent topic:** [Using the OpenCV Template Application](../SDK_tasks/sdk_t_opencv_template.html) **Related information** [Debugging an OpenCV Application](../SDK_tasks/task_runninganopencvapplicationonatarget.html)
