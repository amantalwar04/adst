# Using the Board Support Package Settings Page The **Board Support Package** settings page enables you toconfigure parameters of the OS and its constituent libraries. **Note:** Options for only the libraries that you enabled in the **Overview** page will be visible. Options for the OS/standalone supported peripherals, that are present in the hardware platform, are also shown on the page. ![](../images/sdk_t_config_bsp_internal-2.png) **Parent topic:** [Configuring a Board Support Package \(SDK\)](../SDK_tasks/sdk_t_config_bsp_internal.html) **Related information** [Using the Board Support Package Overview Page](../SDK_tasks/task_usingtheboardsupportpackageoverviewpage.html) [Using the Board Support Package Drivers Page](../SDK_tasks/task_usingthedriverspage.html) [Using the Board Support Package Settings Driver Configuration Page](../SDK_tasks/task_usingthedriverconfigurationpage.html)
