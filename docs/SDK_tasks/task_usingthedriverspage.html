# Using the Board Support Package Drivers Page The **Drivers** page lists all the device drivers assigned for each peripheral in your system. You can select each peripheral and change its default device driver assignment and its version. If you want to remove a driver for a peripheral, assign the driver to **none.** Some device drivers export parameters that you can configure. If a device in the driver list has parameters, it is listed in navigation pane on the left and you can access them by clicking on the device name. ![](../images/sdk_t_config_bsp_internal-3.png) **Parent topic:** [Configuring a Board Support Package \(SDK\)](../SDK_tasks/sdk_t_config_bsp_internal.html) **Related information** [Using the Board Support Package Overview Page](../SDK_tasks/task_usingtheboardsupportpackageoverviewpage.html) [Using the Board Support Package Settings Page](../SDK_tasks/task_usingtheboardsupportpackagesettingspage.html) [Using the Board Support Package Settings Driver Configuration Page](../SDK_tasks/task_usingthedriverconfigurationpage.html)
