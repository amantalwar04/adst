# Creating a New Hardware Platform Specification You can now drag and drop a valid HDF or XML hardware design file to the **Project Explorer** view or drag the system.hdf from one Xilinx SDK instance to another, to create a hardware project. **Note:** Invalid HDF files or files other than HDF or XML hardware design files will result in appropriate error or warning messages in the **SDK Log** view. To create a new hardware project using the **New Hardware Project** wizard, do the following: 1. Click **File \> New \> Project** to open the New Project wizard. 2. Select **Xilinx \> Hardware Platform Specification.** 3. Click **Next.** 4. In the **Project name** field, type a name for the Hardware Platform Specification project. 5. Select the Hardware Platform Specification XML file. **Note:** Hardware Platform Specification XML can be exported by either Project Navigator or VivadoÂ®. ![](../images/sdk_u_new_hardware_platform_spec-1.png) 6.
